# Brick

My boilerplate project for Node.js / React.js development.

* [Express.js](http://expressjs.com/)
* [React.js](https://facebook.github.io/react/)
* [NPM for Everything](http://beletsky.net/2015/04/npm-for-everything.html)

Installation,

```bash
$ npm install
$ npm run build
```
